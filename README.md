# Watchism-repository 观察主义：理学的现代化

## 一、观察主义的形成

在2026年6月的一个普通大课间，我坐在图书馆，手中翻阅着关于朱熹理学的书籍。经过深入的思考和分析，我逐渐领悟到朱熹理学的精髓。朱熹的理学，虽然被归类为客观唯心主义，但我发现，通过重新解释“气”的概念，可以将其与唯物主义联系起来。在这个过程中，我将“理”理解为支配物质世界的规律，而“气”则被视为物质的体现。理与气之间的统一性和相对性，让我认识到，尽管朱熹的理论本质上是唯心的，但通过将“理”纳入一个绝对的“天理”框架，我能够将其转变为一种客观唯心主义的理论。
然而，我开始思考，如果将“天理”这一概念去除，整个理论体系是否可以转变为唯物主义？经过反复推敲，我发现，确实可以将“天理”这一唯心主义的元素剔除，从而构建出一个纯粹的唯物主义理论框架。

## 二、观察主义的基本理论

观察主义，是我对传统理学在现代社会中的一次创新性升级。我将“理”定义为支配物质世界的规律，而“气”则被理解为物质本身。在这个理论中，物质和规律是相互依存、相互制约的。在宏观的物质群体层面上，物质的变化可以被视作规律的体现；而在微观的物质群体中的单个物质层面上，规律的某些方面又可以转化为物质的属性。这种物质与规律的**互约性**，以及它们之间的**相对性**，构成了观察主义理论的核心。

例如，在科学领域，我们可以观察到空间与运动之间的关系。从宏观的角度来看，空间可以被视为一种规律，当物质的尺度发生变化时，空间保持相对不变，但其规律的本质却发生了变化。而在微观视角下，我们所熟悉的水分子、气体等物质，实际上是由可以直接感知的分子构成的。又比如，水的沸点是100摄氏度是规律，这是我们习以为常的常识，但在单个水壶的微观视角下，水的沸点是100摄氏度的规律就会转化为100度的水的物质。此时，如果将100度的水看成规律，那么更微观的视角下，水分子的运动就是物质……以此类推。

## 三、观察主义史观

我所定义的观察主义史观，是一种将历史视为“政策的历史”的观点。在这个史观中，历史的物质基础包括人物、科技等实体，而推动这些实体发展的则是历史的规律和政策。政策本身，可以看作是基于当前历史状况总结出的规律。观察主义史观保持了唯物主义的立场，认为规律在微观个体的视角中会转化为物质，如科技、文化等产物。这些产物又会继续形成新的规律，即新的政策，从而形成一个循环，我称之为**历史的循环向上性**。

历史的循环向上性决定了历史是不断向前发展的。因此，无论是推动物质的改变还是政策的改变，都能推动历史的进程。这就是观察主义史观的核心内容。在使用观察主义史观来评价和编写历史资料时，我们必须注意将物质和政策的作用相互结合。以中国的秦朝为例，物质层面包括秦国的根基和底层的物质条件，而政策层面则包括商鞅变法、统一度量衡等。综合这些因素，我们可以得出秦朝是一个具有复杂影响的朝代，对文化既有积极的一面，也有消极的一面。我不是专业的史学家，对秦朝历史的了解也不够深入，所以我将用我们身边能观察到的政策来说明具体运用观察主义史观的例子。比如对现在的中国教育体系使用观察主义史观，双减等政策是教育部门对当下历史时刻总结出来的规律，但在学校层面，也就是物质层面对于双减的落实并不足，许多学校仍按照原来的规律教学，打断了历史循环向上性的循环。于是，我们就可以得出结论，双减改革不够深化。

## 四、观察主义的意义

我的观察主义，本质上是对传统理学的一种改造和创新。理学原本将一切归结于“理”，使得“理”成为了一种绝对权威。我通过剔除这种权威，并重新解释“相对统一”的概念，提取出“相对性和互约束性”，从而实现了**从理学到观察主义的转变**。这一转变，不仅完成了理论的脱胎换骨，而且将观察主义发展成为了辩证唯物主义的互补。通过这种方式，我希望能够补充中国哲学中的一个缺陷，即理学容易导向主观唯心主义，并最终陷入死胡同的问题。

辩证唯物主义里“物质第一性、规律是物质属性”的固定关系，在面对跨尺度的物理现象时，经常会陷入“到底谁是物质、谁是规律”的模糊地带；而随观察视角互换身份逻辑，刚好能把这些模糊的地方圆上，不用纠结谁是第一性，只看当前观察的尺度。这样，观察主义就成为了一种本体论哲学。比如光速，宏观视角的光速不变支配整个宇宙，当放在一个光子的微观视角上时，就变成了光子的固有属性。所以，**观察主义是辩证唯物主义的发展**。
